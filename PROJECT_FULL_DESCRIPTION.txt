## Executive Summary

UniConsulting is a **production-grade SaaS CRM platform** designed for university consulting agencies to manage international student applications. The platform leverages **Google Gemini 2.0 AI** for document processing, essay review, and university matching, while providing **real-time collaboration** through Supabase Realtime.

> **Core Value Proposition:** "AI-powered university application management that reduces consultant workload by 60% while improving student experience."

---

## Problem Statement

### For Students
- Complex document requirements (transcripts, standardized tests, passports)
- Multiple essay drafts with no structured AI feedback
- Scattered communication with consultants via WhatsApp/Email
- No visibility into application progress

### For Consultants
- Managing 50+ students across different application stages manually
- Repetitive document verification and data entry tasks
- Inconsistent communication tracking across platforms
- Time-consuming university research for each student

## Database Schema (Key Tables)

| Table | Purpose |
|-------|---------|
| `profiles` | User profiles (students + teachers) |
| `student_documents` | Uploaded documents with metadata |
| `essays` | Essay drafts and AI feedback |
| `essay_versions` | Version history for essays |
| `student_universities` | University applications |
| `conversations` | Chat conversation threads |
| `messages` | Individual chat messages |

**Row Level Security (RLS):** All tables protected with policies for:
- Students can only access their own data
- Teachers can access their assigned students
- Admins have full access

- **30-minute gcTime** for persistent cache
- **Background refetch** on window focus
- **Smart prefetching** on link hover

### Context Providers
- `AuthContext` - Session management
- `StudentDataContext` - Student-specific data
- `TeacherDataContext` - Teacher-specific data
- `LanguageContext` - i18n strings
