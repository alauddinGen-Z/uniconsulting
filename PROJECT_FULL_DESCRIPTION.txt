# UniConsulting - MVP Product Description

## Executive Summary
UniConsulting is a **SaaS CRM platform** that helps university consulting agencies manage international student applications efficiently through AI-powered automation.

---

## Problem Statement
International students seeking admission to foreign universities face:
- Complex document requirements (transcripts, test scores, passports)
- Multiple essay drafts with no structured feedback
- Scattered communication with consultants
- Manual, error-prone data entry

Consultants struggle with:
- Managing 50+ students across different application stages
- Repetitive document verification tasks
- Inconsistent communication tracking

---

## Solution: UniConsulting MVP

### Core Value Proposition
**"AI-powered university application management that reduces consultant workload by 60% while improving student experience."**

---

## MVP Features

### For Students
| Feature | Description |
|---------|-------------|
| **Document Upload + AI OCR** | Upload passport, transcripts, IELTS, SAT - AI extracts scores automatically |
| **Essay Writing + AI Review** | Write essays, get instant AI feedback on grammar, clarity, structure |
| **1:1 Consultant Chat** | Direct messaging with assigned consultant |
| **Application Tracker** | View university list and application status |

---

## Technology Highlights
- **AI-Powered**: Google Gemini 2.0 for document OCR, essay review, university matching
- **Real-time**: Live chat and instant updates via Supabase Realtime
- **Multilingual**: English, Russian, Kyrgyz support built-in
- **White-label Ready**: Customizable branding (colors, logo, name)

---

## Target Users
1. **Primary**: Small-to-medium university consulting agencies (5-50 consultants)
2. **Secondary**: Independent education consultants
3. **End Users**: International students (primarily from CIS countries)

---

## MVP Metrics (Success Criteria)
| Metric | Target |
|--------|--------|
| Document processing time | < 30 seconds (vs 5+ minutes manual) |
| Student onboarding | < 10 minutes |
| Consultant efficiency | Manage 30% more students |
| User satisfaction | NPS > 40 |
