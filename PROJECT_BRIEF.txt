           AI-Powered University Application Management Platform
================================================================================

OVERVIEW
--------
UniConsulting is a comprehensive SaaS platform that connects educational 
consultants with students to streamline the university application process.
Built with modern web technologies and AI capabilities, it provides tools
for managing applications, reviewing essays, and automating form filling.


KEY FEATURES
------------
• Student Dashboard     - Track applications, upload documents, manage essays
• Teacher Dashboard     - Manage student queue, Kanban board, approve accounts
• AI Essay Review       - Google Gemini-powered feedback on college essays
• AI University Matcher - Intelligent school recommendations based on profile
• Document OCR          - Passport/transcript scanning with Vision API
• Real-time Chat        - Supabase-powered messaging between users
• Desktop App           - Windows .exe with Python automation for forms


TECH STACK
----------
Frontend:   Next.js 16, React 19, Tailwind CSS
Backend:    Supabase (Auth, Database, Realtime, Storage)
AI:         Google Gemini 2.0 Flash
Desktop:    Electron + Python (browser-use library)
Deployment: Netlify


TARGET USERS
------------
• Educational consultants managing multiple students
• International students applying to universities abroad
