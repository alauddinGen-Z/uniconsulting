================================================================================
                           UNICONSULTING
           AI-Powered Study Abroad Consulting Platform
================================================================================

OVERVIEW
--------
UniConsulting is a modern web application designed for study abroad consulting 
agencies. It streamlines the entire student application process with AI-powered 
tools, real-time collaboration, and an intuitive dashboard experience.

CORE FEATURES
-------------
• Student Dashboard - Complete application management portal
• Teacher Dashboard - Student tracking with Kanban boards
• AI Document OCR - Automatic data extraction from passports, IELTS, SAT, GPA
• AI Essay Review - Instant feedback and coaching for personal statements
• AI University Matcher - Smart recommendations based on student profiles
• Real-time Chat - Direct messaging between students and teachers
• Document Management - Upload, organize, and track all application documents

AI CAPABILITIES
---------------
1. Document OCR (Google Gemini)
   - Passport: Extracts name, DOB, passport number, nationality, expiry
   - IELTS: Extracts all band scores (Overall, Listening, Reading, Writing, Speaking)
   - SAT: Extracts total score and section scores
   - GPA/Transcript: Extracts yearly GPAs and cumulative scores

2. Essay Review (Google Gemini)
   - Overall scoring (1-10)
   - Strengths and improvement areas
   - Grammar corrections
   - "Show don't tell" analysis
   - Coaching prompts for revision

3. University Matcher (Google Gemini)
   - Analyzes student profile and scores
   - Recommends matching universities
   - Considers budget, location, and program preferences

TECHNOLOGY STACK
----------------
• Frontend: Next.js 16, React 18, TypeScript
• Styling: TailwindCSS, Framer Motion
• Backend: Supabase (PostgreSQL, Auth, Realtime, Storage)
• AI: Google Gemini 2.0 Flash via Supabase Edge Functions
• Hosting: Netlify (Frontend), Supabase (Backend)

USER ROLES
----------
1. STUDENT
   - Fill profile and upload documents
   - Write and get AI feedback on essays
   - Track university applications
   - Chat with assigned teacher

2. TEACHER
   - Approve/reject new student registrations
   - View and manage all assigned students
   - Track student progress via Kanban board
   - Communicate via real-time chat

3. ADMIN
   - Add new teacher accounts
   - Full system access

KEY PERFORMANCE FEATURES
------------------------
• Instant tab switching (no loading spinners)
• Real-time data updates via Supabase subscriptions
• Image compression for faster OCR processing
• Global state management for seamless navigation

DEPLOYMENT
----------
Frontend: Netlify (auto-deploys from GitHub)
Backend: Supabase Cloud
Edge Functions: Supabase Edge (Deno runtime)
