UNICONSULTING: EdTech CRM for university application consulting.
Students upload docs (IELTS/TOEFL/GPA/Passport), write essays, chat with teachers.
Teachers approve students, manage Kanban pipeline, review applications.
Tech: Next.js 16, React 19, Supabase, TailwindCSS, TypeScript.

Features: Student approval workflow, document management, profile tabs, 
Kanban board, real-time chat, AI Essay Review (Gemini), form automation.
